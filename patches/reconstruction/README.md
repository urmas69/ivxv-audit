# Controlled reconstruction patches

`0001-use-published-local-core.patch` is the only source patch currently
applied. It replaces the inaccessible `tivi.io/core` pseudo-version with the
published root `core/` snapshot. The replacement is an experiment for build
research, not an assertion about KOV2025 production inputs. The exact source
commit, URL, hash, license, and behavioral equivalence of the pseudo-version
remain unresolved; see `reports/reconstructed-build.md`.
